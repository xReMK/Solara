The File Format: GGUF - GPT-Generated Unified Format

A GGUF file is divided into three sequential sections:
	Header: A small magic number identifying the file and the version.
	Metadata (The Config): Key-value pairs containing the "blueprints"—model architecture (Gemma, Llama), number of layers, attention heads, context length, and the Tokenizer vocabulary (the map of text strings to ID numbers).
	Tensors (The Weights): The bulk of the 10GB. These are huge multi-dimensional arrays of numbers. In your 4-bit model, these are "packed" integers
